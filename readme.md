# 🎧 Spotify Data Analysis Dashboard

## 🚀 Project Overview
This project is an interactive Streamlit dashboard that analyzes Spotify music dataset.  
It provides insights into songs, artists, genres, and audio features like energy and danceability.

---

## 📂 Features

- 📄 Dataset preview
- 🧹 Data cleaning (missing values handling)
- 🎧 Top songs analysis
- 🎤 Artist performance analysis
- 📊 Genre distribution
- 📈 Popularity trends
- 🔥 Audio feature analysis (energy, danceability, loudness)
- 📊 Interactive visualizations

---

## 🛠️ Technologies Used

- Python
- Streamlit
- Pandas
- NumPy
- Plotly
- Matplotlib
- Seaborn
